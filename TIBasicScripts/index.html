(Somewhat) useful TI-BASIC scripts

These scripts are written for a TI-85, so there are modern counterparts to many of these commands.
-> represents the STO> key.

BASECONV

A Base Converter that works from bases 2-10. Other bases are undefined behavior. 

ClLCD

Input "DIGITS: ",D
Input "NUM: ",A
Input "SRC BASE: ",B
Input "DEST BASE: ",F

O->T
O->C

For(J,0,D-1)
int(A/(10^(D-1-J)))->C
C*B^(D-1-J)+T->T
A-C*10^(D-1-J)->A
End

1->P
0->E
For(K,0,15)
int(T/(F^(15-K)))->E
Outpt(5,P,E)
T-(E*F^(15-K))->T
P+1->P
End
Disp ""
BASELOG

A calculator that solves logarithmic expressions of form logB(A) = E 

Disp "BASE - 1"
Disp "ARGUMENT - 2"
Disp "EXPONENT - 3"
Disp "WHICH NEEDED? "
Prompt Z

If(Z==1)
Then
Prompt A
Prompt E
Disp (E^x√A)
End

If(Z==2)
Then
Prompt B
Prompt E
Disp (B^E)
End

If(Z==3)
Then
Prompt B
Prompt A
log A->T1
log B->T2
Disp (T1/T2)
End
BOUNCE

An animation that bounces a pixel across your graph screen 

ClLCD
ZStd
ZSqr
AxesOff
DispG

0->A
0->B
e^(-1)->C
π^(-1)->D

While (1==1)
If A>=17 or A<=-17
C*-1->C
If B>=10 or B<=10
D*-1->D
A+C->A
B+D->B
PtChg(A,B)
End
